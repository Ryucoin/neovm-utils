### OEP4 Class

The `OEP4Interface` class is used to interact with [OEP4](https://github.com/ontio/OEPs/blob/master/OEPS/OEP-4.mediawiki) compliant smart contacts.

``` swift
public class OEP4Interface: OEP10Interface
```

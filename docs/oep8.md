### OEP8 Class

The `OEP8Interface` class is used to interact with [OEP8](https://github.com/ontio/OEPs/blob/master/OEPS/OEP-8.mediawiki) compliant smart contacts.

``` swift
public class OEP8Interface: OEP10Interface
```
